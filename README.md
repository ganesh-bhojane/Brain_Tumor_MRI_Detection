# Brain Tumor MRI Classification

## Overview
Deep learning project to classify MRI brain scans into different tumor categories using MobileNetV2.

## Dataset
Brain Tumor MRI Dataset from Kaggle.

## Technologies
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Matplotlib

## Model
MobileNetV2 Transfer Learning

## Results
- Training Accuracy: 89%
- Validation Accuracy: 91%
## Deployment

This project is deployed using Gradio.

Users can upload an MRI image and receive a predicted tumor class.

### Features
- Upload MRI scan
- Real-time prediction
- Deep Learning model (MobileNetV2)
- Simple web interface using Gradio
